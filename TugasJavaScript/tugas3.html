FAQ Interaktif

Apa itu jQuery? ▶
jQuery adalah library JavaScript yang mempermudah manipulasi DOM, event handling, dan animasi pada website.
Bagaimana cara menggunakan .click() di jQuery? ▶
Metode .click() digunakan untuk menangani event klik pada elemen HTML dengan cara yang lebih sederhana.
Apa fungsi dari .toggle()? ▶
.toggle() digunakan untuk menampilkan dan menyembunyikan elemen secara bergantian.
Apa perbedaan antara .hover() dan CSS :hover? ▶
CSS :hover hanya untuk efek visual, sedangkan .hover() di jQuery bisa menjalankan fungsi JavaScript.
Apakah jQuery masih relevan di tahun ini? ▶
Masih relevan untuk proyek sederhana, meskipun framework modern seperti React dan Vue lebih sering digunakan.
